# Manipulación de Datos (Data Wrangling) - Parte I

## Descripción
Proyecto enfocado en técnicas de **limpieza y transformación de datos**.  
Se aplican operaciones comunes como selección, filtrado, agrupación y ordenamiento.

## Objetivos
- Dominar funciones esenciales de pandas.
- Transformar datos para análisis estadístico.
- Resolver problemas de datos faltantes y duplicados.

## Tecnologías
- Python
- pandas

## Resultados Clave
- Limpieza efectiva de datasets.
- Identificación y manejo de valores atípicos.
- Tablas listas para análisis exploratorio.
